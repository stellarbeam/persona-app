# persona

A Flutter project which uses BLoC (Business Logic Component) architecture to build an app to demonstrate login flow.

It has these features:  
 - OTP based login [uses BLoC and Firebase]
 - Light/Dark theme, with animated change [uses BLoC]
    - can be changed from anywhere in login process
 - Multiple language support
    - can be changed from anywhere in login process
    - currently supports English and Hindi only, contributers are welcome to contribute translations in other languages
 - Onboarding flow
    - pages with beautiful drawing-type animations [credit to LottieFiles]
    - also smooth transitions between pages
